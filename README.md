# 使用 TestCafe 測試前端
本範例將使用 TestCafe 來測試前端，其中以 React 開發的 To Do List 為例。

![img](/doc/doc.gif)

## 版本資訊

* Node: v8.11.4

## 安裝

```
yarn
```

或

```
npm i
```

## 測試

1. 執行 App
```
npm start
```

2. 待 App 啟動成功，則可開新終端機執行下列指令
```
testcafe chrome src/App.e2e.js
```

## 啟動 App

```
npm start
```
